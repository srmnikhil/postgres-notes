# Basics

## What is a Database?

An organised collection of Data.
A method to manipulate and access the data.

## Database vs DBMS

### Database

- A database is a structured collection of data.
- It stores information in tables or files.
- It cannot perform operations by itself.
- It is passive and just holds data.

Examples:

- Users table
- Orders records
- Product list

In simple terms:

> Database is the actual stored data.

---

### DBMS

- DBMS stands for Database Management System.
- It is software used to manage databases.
- It allows creating, reading, updating, and deleting data.
- It provides security, concurrency, and backup features.

Examples of DBMS:

- PostgreSQL
- MySQL
- Oracle
- SQL Server

In simple terms:

> DBMS is the software that manages the data.

## RDBMS

A type of database system that stores data in structed tables (using rows and columns) and uses SQL for managing and querying data.

## SQL vs PostgreSQL

### SQL

- Structured Query Language, Which is used to talk to our database.
- Example :- `SELECT * FROM users;`

### PostgreSQL

- PostgreSQL is an open-source relational database management system (RDBMS).
- It uses SQL as its primary query language.
- It actually stores the data on disk.
- It provides features like indexing, transactions, and security.

In simple terms:

SQL is the language.
PostgreSQL is the software that understands and executes that language.
